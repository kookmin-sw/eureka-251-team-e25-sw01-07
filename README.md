### <정진우 선배님 Interview> <br>

#### Q1. 공부하고 계신 분야에 대해 더 공부하고 싶다고 생각하게 된 계기가 무엇인가요?

A1. 학부연구생으로 연구실에 들어가면서 본격적으로 인공지능을 접하게 되었습니다. 논문을 읽고, 내용을 공부한 뒤, 논문에 제시된 모델을 직접 구현해서 돌려 보면서 성능을 높이는 아이디어를 적용해 보는 과정이 재미있었습니다. 인공지능 분야에서 제대로 논문을 공부하고 논문을 쓰기 위해서는 연구실에서 체계적으로 연구를 해야겠다 느꼈고, 대학원을 진학하게 되었습니다.

#### Q2. 대학원에서 어떤 공부(혹은 연구)를 하고 계신가요?
